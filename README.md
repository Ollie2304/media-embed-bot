# Media Embed Bot

Automatically embeds Twitter / X, Reddit & TikTok media in Discord.

## Installation

`npm` package manager

```
npm install
```

Discord bot token is stored in a `.env` in `./` with the format:
```dotenv
TOKEN="token"
CLIENT_ID="client id"
```
