# Media Embed Bot

Automatically embeds Twitter / X, Reddit & TikTok media in Discord.

## Installation

`npm` package manager

```
npm install
```

Discord bot token is stored in a `config.json` in `src` with the format:
```json
{
  "token": "TOKEN",
  "clientId": "CLIENTID"
}
```
