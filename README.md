# Media Embed Bot

Automatically embeds:
- Twitter / X
- Reddit
- Instagram
into discord chat

>requires webhook perms as it creates a webhook to "fake" looking like the user posted it.

## Installation

`npm` package manager

```
npm install
```

Discord bot token is stored in a `.env` in `./` with the format:
```dotenv
TOKEN="token"
CLIENT_ID="client id"
```
