# Media Embed Bot

Automatically embeds [Twitter / X](https://github.com/FixTweet/FixTweet), Reddit & [TikTok](https://github.com/dylanpdx/vxtiktok) media in Discord.

## Installation

`npm` package manager

```
npm install
```

Discord bot token is stored in a `config.json` in `src`
